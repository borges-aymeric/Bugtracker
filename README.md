# BUGTRACKER

Application de suivi de bugs (tickets) en PHP avec MySQL.

## Prérequis

- PHP
- MySQL
- Base de données : importer `sae203.sql`

## Installation

1. Cloner le dépôt
2. Créer la base de données et importer `sae203.sql`
3. Configurer la connexion dans `connexion.php`

## Structure

- **Authentification** : `base_authentification.html`, `connexion.php`, `deconnexion.php`, `login.php`
- **Tickets** : `tickets.php`, `ticketadd.php`
- **Référentiels** : `priority.php`, `statut.php`, `developper.php`
